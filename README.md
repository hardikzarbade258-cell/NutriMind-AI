# NutriMind AI 
AI-Powered Smart Nutrition & Decision Engine

##  Overview
NutriMind AI is an intelligent diet planning system that generates personalized meal plans based on user goals, pantry availability, health conditions, and budget constraints.

Unlike traditional diet apps, this system uses a rule-based AI engine combined with decision logic to create adaptive, real-world meal plans.

---

##  Key Features

- AI-driven meal plan generation  
- Pantry-based smart recommendations  
- Budget-aware diet planning  
- Macro & calorie optimization per meal  
- Health condition filtering (Diabetes, BP, PCOS, etc.)  
- Allergy-aware food filtering  
- Smart fallback logic (no-match scenarios handled)  
- Recipe scoring & ranking system  
- Automatic shopping list generation  
- Multi-profile support system  
- Admin control panel  

---

##  How It Works

1. User inputs:
   - Age, weight, goal (fat loss / gain / maintenance)
   - Food preferences (veg / non-veg)
   - Pantry items
   - Budget
   - Health conditions

2. AI Engine:
   - Filters recipes based on constraints
   - Scores meals using nutrition + cost + relevance
   - Applies optimization logic

3. Output:
   - Balanced meal plan
   - Nutrition breakdown
   - Missing ingredients
   - Fix suggestions
